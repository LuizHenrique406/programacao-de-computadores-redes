# Linguagens de alto e baixo nível

São linguagens que ficam entre a **linguagem da máquina** e a dos **humanos**.
- **Alto nível:** Java, Python, JavaScript
- **Baixo nível:** Assembly

> As de baixo nível são aquelas que utilizam números binários, 0 ou 1 e tê contato quase direto com o hardware

> As de alto nível usam uma lógica mais abstrata, sendo possível declarar métodos, variáveis, criar laços, etc.
