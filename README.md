# Programacao de Computadores
## Linguagem de alto e baixo nível
São linguagens que ficam entre a linguagem da máquina e a dos humanos.
- alto nível: Java, Python, JavaScript...
- baixo nível: Assembly...

As de baixo nível são aquelas que utilizam números binários, 0 ou 1.

As de alto nível usam uma lógica mais abstrata.
